# Rule: Plan Mode

## Planning Instructions

1. Start by naming the active phase.
2. Use `docs/project-phase-plan.md` and the matching `.bob/plans/*` file.
3. Use `docs/langgraph-architecture.md` as the architecture map for cross-layer work.
4. Keep plans short and executable.
5. Include goal, scope, files, checkpoints, and risks.
6. Do not add external workflow automation, CRM, payment, booking, or outbound campaigns unless explicitly requested.
7. End with a clear next action.

## Output Pattern

- Active phase
- Goal
- Scope
- Files
- Checkpoints
- Risks
- Next action
