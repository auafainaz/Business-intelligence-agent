# Phase 1 - Frontend Plan

## Goal

Build the premium public website and dashboard UI for ClientIQ.

## Stack

- Next.js
- JavaScript only
- Tailwind CSS

## Scope

- homepage
- about/how-it-works content if needed
- premium dark Apple-like design
- strong hero
- visible Twilio phone CTA
- feature sections
- dashboard preview
- `/dashboard/[client-slug]` route UI
- graceful missing/partial dashboard states
- responsive desktop/mobile layout

## Excludes

- Twilio backend calls
- xAI/Grok realtime logic
- SQLite persistence
- Gmail/SMS delivery
- backend orchestration

## Design Direction

- premium, calm, modern
- dark theme
- strong typography
- concise copy
- no clutter
- phone number visible early
- dashboard page useful even with partial data

## Done When

- frontend starts locally
- homepage CTA is clear
- dashboard page renders with valid and invalid slug states
- responsive layout works
